Bachelor's Thesis (Ongoing)

RL-Based Precision Drone Landing on Moving Platforms

My thesis focuses on training a quadcopter to autonomously land on a moving platform using deep reinforcement learning. The agent learns to track the platform, compensate for disturbances, and execute precise touchdowns.

Current Progress

  • PPO and SAC models converged successfully
  • PPO shows higher reliability in landing accuracy
  • Implementing Control Barrier Functions for safety guarantees
  • Real drone testing after simulation validation

RL Architecture

PPO with custom reward shaping for landing precision, velocity matching, and stability. SAC tested for comparison.

Safety Layer

Implementing Runtime Assurance using Control Barrier Functions based on research by Hobbs, Mote, Abate, Coogan & Fero.

Training Metrics

Multiple PPO and SAC runs compared. PPO_10 (pink) shows best convergence with stable episode lengths and reward around 60.

Akkodis Service Robot

Autonomous Object Detection & Retrieval System

RViz2 showing Nav2 navigation with real-time YOLO object detection

TurtleBot3 autonomous object retrieval demo

Enhanced a TurtleBot3 with advanced 3D perception using a RealSense D405 depth camera and YOLO object detection. The robot can identify and locate objects in 3D space for autonomous retrieval tasks.

Technical Architecture

  • Distributed ROS2: Pi 5 (Jazzy) + Workstation (Humble)
  • Custom ZeroMQ Bridge: Camera data + YOLO inference transfer
  • Eclipse Cyclone DDS: Low-latency real-time operation
  • 3D Perception: RealSense D405 depth camera

Pixhawk Drone Platform

Custom UAV for Autonomous Research

Building a custom Pixhawk-based drone platform for my thesis research on autonomous landing. The drone serves as the physical testbed for validating RL policies trained in simulation.

Platform Specifications

  • Current Status: Hardware assembly complete, debugging motor calibration
