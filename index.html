VR Accident Reconstruction

Dash-Cam & Simulation for Forensic VR Reconstruction

3D Laser-Scanned Scene
Simulation Environment

Project Overview

Bridged real-world evidence (dashcam footage + FARO Focus Core 3D laser scans) with driving simulators for forensic VR accident reconstruction. Imported high-fidelity point cloud environments into IPG CarMaker, integrated a Skoda Octavia vehicle model, and aligned virtual scenarios with real-world coordinates via OpenStreetMap data. Also explored CARLA as a parallel digital twin platform.

Key Contributions

  • 3D Scene Capture: Acquired point cloud data with FARO Focus Core laser scanner (±2 mm accuracy)
  • Simulation Pipeline: Imported scanned 3D environments into IPG CarMaker for driving simulation
  • Vehicle Integration: Added Skoda Octavia model with dashcam perspective for scenario replay
  • Geo-Alignment: Aligned virtual scenario with real-world coordinates via OpenStreetMap
  • CARLA Digital Twin: Explored open-source CARLA simulator as an alternative reconstruction platform
